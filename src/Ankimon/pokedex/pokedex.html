PokéDex
Welcome to the PokéDex!

INSTALL APP

SETTINGS

Mega Evolution Animation

If enabled, the animation takes place once for each species after the base form's image has been downloaded.

Save Selected Games

If enabled, the selected game version is saved, else the latest version is used. Game versions affect moves and ability flavor texts.

ABOUT

v2.0
The current PokéDex is up to date with the data from the PokéAPI (Uptu Gen VIII). Earlier in Development this app used to call the API but now all the data is completely stored locally to facilitate offline usage. The PokéAPI database is accurate but its not perfect. Therefore, some data might be inconsistent.
Made with for Pokémon.
