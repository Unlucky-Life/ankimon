Read Like a Pro

Understand the existing code of Ankimon

‚è±Ô∏è 1 hour Medium ‚ñº

Don't Panic!

It's okay if you don't understand everything at first. Focus on getting a general idea of how things work. AI helps with the rest!

Explore Ankimon Functions

In VS Code, open Ankimon code, go to Explorer from the left tab (Ctrl+Shift+E), and navigate to the src/Ankimon/functions directory

Try to Understand a File

Open any file in that directory, and check out one of its functions. Use help from AI. Can you understand what the function is trying to do? What does every line do?

Practice Code Reading and Understanding

Find three functions in the file, and in your own words, explain what the functions do. Again, AI help is allowed!

See How It All Connects

Take a look at the src/Ankimon/__init__.py file. This is the main file that brings everything together. Notice how it imports and uses code from other files you've seen. Try to follow one or two function calls to see where they lead!

AI-Assisted Coding

Start coding with the help of AI

‚è±Ô∏è 1 hour Medium ‚ñº

Find a Bug to Fix

Go on the Ankimon Discord in the bugs-and-issues channel to find a bug to fix!

Prompt Designing and AI Assistance

Ask AI to help you fix the issue! Make a prompt that ensures that the AI will explain to you the errors clearly, and help make the code fixes properly. Make sure you understand the code changes!

Error Debugging

Your AI is a powerful tool, but it can make mistakes. When you see an error, don't get discouraged! Read the error message carefully. A good practice is to copy the full error and ask your AI 'What does this error mean and how can I fix it?'. This teaches you how to solve problems like a real developer.
